pl2js — Prolog Browser Editor

Translate and run Prolog in your browser — no installation needed.

  1. Type or paste a Prolog program into the editor, or click Load .pl to open a file.
  2. Enter a query in the query field, e.g. parent(X, Y).
  3. Click Run Query to see results.
  4. Click Save .pl to download the current program. The editor remembers your text between visits. If you have unsaved changes, the page will warn you before leaving.
